# AnyLanguageExpressaic — Roadmap

## Goal

Make it easy to understand programming concepts across multiple languages with a focus on thinking, not syntax.

## Current

* Basic GUI renderer with Table, Code, Visual, and YAML views
* YAML-based concept structure
* Multi-language comparison support

## Next

* Add more core concepts (loops, functions, data structures, etc.)
* Improve comparison clarity and visuals
* Enhance YAML editing experience
* Add more languages, Image support

## Planned

* CLI tools for generating and validating YAML
* Better search and navigation across concepts
* Export to Markdown or documentation formats
* Improve UI and overall usability

## Distribution

* Build standalone executable
* Add icon and branding
* Release via GitHub

## Future

* VS Code extension
* Web version
* Intelligent features

## Contributions

* Add new concepts in YAML format
* Improve existing mappings
* Suggest better comparisons
* Suggest better ways to integrate with existing software / programming environments

## Vision

Learn a concept once and apply it across languages.
